\begin{abstract}[zh]
工业过程故障诊断对于保障生产安全、降低非计划停机损失和提升过程运行可靠性具有重要意义。然而，流程工业在不同操作条件、负载水平和运行阶段下常伴随明显的分布偏移，使得仅利用源域标注数据训练得到的模型在目标域上的诊断性能显著下降。围绕这一问题，本文以 Tennessee Eastman Process Domain Adaptation 数据集为研究对象，研究单源到单目标的无监督领域自适应故障诊断，并在统一实验设定下比较不同领域自适应方法。

本文在统一的一维卷积骨干网络下，对 Source Only、CORAL、DAN、DANN、CDAN 和 DeepJDOT 等代表性方法进行了系统比较。针对流程工业多变量时序信号的特点，本文统一了域划分方式、样本表示、归一化策略、训练协议、模型选择准则和评价指标，并采用准确率、平衡准确率、混淆矩阵与特征可视化等方式开展分析。六组代表性单源场景及部分多源场景结果表明，相较于 Source Only，分布对齐和对抗式方法在目标域上普遍能够取得明显提升，其中 DAN 与 DANN 的平均表现较优，但不同场景间仍存在稳定性差异和类别混叠现象。

在基线分析基础上，本文进一步提出一种可靠性门控的类条件时序自适应思路，通过结合教师网络伪标签、可靠性筛选、类别原型约束与一致性正则，增强跨域同类聚合能力并抑制噪声伪标签带来的负迁移影响。尽管全量场景统计和改进方法实验结果仍有待补充，本文已经完成了论文整体结构、实验基线与改进方法设计，并形成了较为清晰的流程工业领域自适应故障诊断研究主线。

\keywords{故障诊断, 领域自适应, TEP, 无监督领域自适应}
\end{abstract}

\begin{abstract}[en]
Industrial fault diagnosis is important for production safety, reduction of unplanned downtime, and reliable process operation. However, process data collected under different operating modes, load levels, or time periods usually exhibit evident distribution shifts, so a model trained only with labeled source-domain data often generalizes poorly to a target domain. To address this issue, this thesis studies single-source to single-target unsupervised domain adaptation for fault diagnosis on the Tennessee Eastman Process Domain Adaptation dataset and compares representative adaptation methods under a unified experimental protocol.

Under a unified one-dimensional convolutional backbone, six representative methods are compared: Source Only, CORAL, DAN, DANN, CDAN, and DeepJDOT. To make the comparison consistent and reproducible, this work uses a common setting for domain partition, input representation, normalization, training protocol, model selection, and evaluation, together with accuracy, balanced accuracy, confusion matrices, and feature visualization.

Results on six representative single-source scenarios together with part of the multi-source settings show that, compared with the Source Only baseline, distribution-alignment and adversarial methods generally improve target-domain accuracy, while DAN and DANN achieve stronger average performance among the completed scenes. The experiments also reveal remaining issues such as scenario sensitivity, class confusion, and unstable conditional alignment.

Based on these observations, the thesis further develops a reliability-gated class-conditional temporal adaptation strategy. It combines teacher-generated pseudo labels, reliability screening, prototype-based constraints, and consistency regularization to improve cross-domain intra-class compactness while reducing the effect of noisy target supervision. Although full-scenario statistics and final validation still need to be added, the current draft already establishes a clear comparative baseline and a concrete method-design direction.

\end{abstract}
