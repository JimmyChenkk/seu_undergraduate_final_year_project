\begin{abstract}[zh]
工业过程故障诊断对于保障生产安全、降低非计划停机损失和提升过程运行可靠性具有重要意义。然而，流程工业在不同操作条件、负载水平和运行阶段下常伴随明显的分布偏移，使得仅利用源域标注数据训练得到的模型在目标域上的诊断性能显著下降。围绕这一问题，本文以 Tennessee Eastman Process Domain Adaptation 数据集为研究对象，研究单源到单目标的无监督领域自适应故障诊断，并构建面向 benchmark 复现与方法比较的实验工作流。

本文结合当前项目实现，完成了数据勘察、基准元数据构建、训练评估、结果汇总与图表导出等基础模块，并在统一的一维卷积骨干网络下实现了 Source Only、CORAL、DAN、DANN、CDAN 和 DeepJDOT 等代表性方法。针对流程工业时序信号的特点，本文统一了域内标准化、固定折划分、目标域无标签训练和目标域测试评估协议，并采用准确率、平衡准确率、混淆矩阵与特征可视化等指标开展分析。六组代表性单源场景及部分多源场景结果表明，相较于 Source Only，分布对齐和对抗式方法在目标域上普遍能够取得明显提升，其中 DAN 与 DANN 的平均表现较优，但不同场景间仍存在稳定性差异和类别混叠现象。

在 benchmark 分析基础上，本文进一步提出一种原型引导的类条件对齐思路，通过结合源域类别原型、高置信目标域伪标签与全局域对齐损失，增强跨域同类聚合能力。尽管全量场景统计和改进方法实验结果仍有待补充，本文已经完成了论文整体结构、实验基线与改进方法设计，并形成了一个较为规范、可复现的流程工业领域自适应故障诊断实验框架。

\keywords{故障诊断, 领域自适应, TEP, 基准实验}
\end{abstract}

\begin{abstract}[en]
Industrial fault diagnosis is important for production safety, reduction of unplanned downtime, and reliable process operation. However, process data collected under different operating modes, load levels, or time periods usually exhibit evident distribution shifts, so a model trained only with labeled source-domain data often generalizes poorly to a target domain. To address this issue, this thesis studies single-source to single-target unsupervised domain adaptation for fault diagnosis on the Tennessee Eastman Process Domain Adaptation dataset and builds a benchmark-oriented workflow for method comparison.

The implemented pipeline covers data inspection, benchmark metadata construction, model training, evaluation, result aggregation, and figure export. Under a unified one-dimensional convolutional backbone, six representative methods are compared: Source Only, CORAL, DAN, DANN, CDAN, and DeepJDOT. To make the benchmark comparable and reproducible, this work uses a consistent protocol for domain-wise normalization, fixed fold splitting, unlabeled target-domain adaptation, and target-domain evaluation, together with accuracy, balanced accuracy, confusion matrices, and feature visualization.

Results on six representative single-source scenarios together with part of the multi-source settings show that, compared with the Source Only baseline, distribution-alignment and adversarial methods generally improve target-domain accuracy, while DAN and DANN achieve stronger average performance among the completed scenes. The benchmark also reveals remaining issues such as scenario sensitivity, class confusion, and unstable conditional alignment.

Based on these observations, the thesis formulates a prototype-guided class-conditional adaptation strategy. It combines source prototypes, high-confidence target pseudo labels, and global domain alignment to improve cross-domain intra-class compactness. Although full-scenario statistics and final validation still need to be added, the current draft already establishes a reproducible benchmark baseline and a concrete method design.

\end{abstract}
